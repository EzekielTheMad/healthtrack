# Contributing

Issues and pull requests are welcome.

## Getting started

```bash
npm install
npm run dev
```

## Before submitting a PR

- Run the test suite: `npm test` (and `npm run lint`, `npm run build`).
- Add or adapt tests for behavior changes.
- Use [conventional commits](https://www.conventionalcommits.org/) (`feat:`, `fix:`, `chore:`, ...).

Note: this project uses Next.js 16, which differs in places from older Next.js conventions — see `AGENTS.md` and the docs bundled in `node_modules/next/dist/docs/` when in doubt.
